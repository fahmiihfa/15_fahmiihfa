BELAJAR CSS

Apa Itu CSS?

CSS adalah bahasa Cascading Style Sheet dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.
Sintaks CSS terdiri dari tiga bagian: pemilih atau selektor (selector), sifat atau properti (property), dan nilai (value). “Selektor” biasanya adalah elemen atau tag HTML yang akan didefinisikan, “properti” adalah atribut yang akan diganti dengan “nilai” tertentu.

Bagaimana Menggunakan CSS?

Ada 3 cara menggunakan CSS:

  1. Inline CSS: digunakan untuk memberi style (gaya) pada tag atau elemen HTML tertentu.
  2. Internal CSS: digunakan untuk memberi style yang hanya berlaku pada satu laman web.
  3. External CSS: adalah file berekstensi .css yang hanya berisi kode-kode style.
